Interwheel — A* agent

A search-tree planner for the climbing blob, in the spirit of Robin Baumgarten's 2009 Mario A* agent. Every replan, the planner forward-simulates many candidate futures from a snapshot of the live game; the trees you see overlaid on the wheels are those candidate futures, with the chosen path drawn brightest.

D toggle the candidate-line overlay.
A toggle the AI on/off (you can take over with Space).
R reload the page for a fresh level.

Live game from src/games/interwheel/index.ts, unchanged gameplay code paths. The AI runs on snapshot clones of the same game instance.
